# Linux From Scratch – Documentação

Este repositório documenta a construção de um sistema Linux
baseado no projeto Linux From Scratch (LFS), seguindo o livro
oficial como fundação técnica.

## Objetivo
Registrar decisões técnicas, problemas encontrados e soluções
aplicadas durante a construção de um sistema LFS utilizável,
sem tratar o projeto apenas como material didático.

## Escopo
- LFS base completo
- Extensões selecionadas (BLFS)
- Uso consciente como sistema diário
- Documentação técnica, não tutorial passo-a-passo

## Ambiente
- Host: Windows
- Virtualização: VirtualBox
- CPU: i5-10400F
- RAM: 16 GB
- Disco da VM: 30 GB
